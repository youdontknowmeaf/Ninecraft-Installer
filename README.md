# Ninecraft-Installer
This is a 3rd party installer for Ninecraft (from MCPIReborn repo)

# Which version should you choose?
Select ***auto-install*** version if you want to **install** the game **fast**, **without** the need to use your **brain**.
Select ***semi-auto*** version if you want to **install** the game with **custom apk**, requires **more** **brain** activity.

# How to install?
Download the installer of your choice, enter the terminal and ``chmod +x /path/to/ninecraft-installer`` then launch the executable as sudo. (``sudo ./ninecraft-installer``)
